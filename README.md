# 💾 18. Backup жүйесі (File Backup & Restore)

## 🎯 Мақсаты:
Бұл жобаның мақсаты — деректерді қорғау және олардың резервтік көшірмесін (backup) жасау. Бағдарлама файлдағы ақпаратты сақтап, қажет болған жағдайда оны қайта қалпына келтіруге (restore) мүмкіндік береді.

---

## 📌 Жобаның сипаттамасы:
Бұл бағдарлама файлдармен жұмыс істейді. Негізгі файлдан деректерді оқып, оның көшірмесін басқа файлға сақтайды (backup). Қажет болған жағдайда сол сақталған көшірмені бастапқы немесе жаңа файлға қайта жүктейді (restore).

---

## ❓ Сұрақтар және жауаптар:

### 1. Backup не үшін қажет?
Backup деректердің қауіпсіздігін сақтау үшін қажет. Егер файл жоғалып кетсе, бұзылса немесе қате өзгертілсе, backup арқылы оны қайта қалпына келтіруге болады.

---

### 2. Қай кезде restore қолданылады?
Restore келесі жағдайларда қолданылады:
- Файл кездейсоқ өшіп кеткенде  
- Деректер бұзылғанда  
- Жүйе істен шыққанда  
- Бұрынғы нұсқаны қайтару қажет болғанда  

---

### 3. Деректер қалай сақталады?
Деректер файл түрінде сақталады (мысалы: `.txt`, `.dat`). Бағдарлама файлды оқып, оны басқа орынға көшіріп резервтік көшірме (backup) жасайды.

---

## 📊 Қорытынды:
Бұл жоба файлдық деректермен жұмыс істеуді үйретеді. Backup және restore функциялары ақпараттың қауіпсіздігін қамтамасыз етеді. Мұндай жүйе деректер жоғалу қаупін азайтып, ақпаратты кез келген уақытта қайта қалпына келтіруге мүмкіндік береді.
